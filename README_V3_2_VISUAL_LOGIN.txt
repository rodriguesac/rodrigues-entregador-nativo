# Rodrigues Entregador V3.2 - 100% nativo

Esta versão mantém o caminho 100% nativo, sem WebView/Capacitor, mas aproxima o visual do PainelUP/Up Entregas.

Inclui:
- visual nativo com identidade PainelUP: fundo escuro, roxo #4B0082, verde #82C91E, cards arredondados;
- login por CPF/telefone e senha quando existir no cadastro;
- entregadores antigos sem senha continuam entrando por CPF/telefone;
- cadastro nativo no app, salvo como PENDENTE em entregadores e solicitacoesEntregadores;
- aprovação continua no painel gestor;
- mantém Firebase real, pedidos reais, som/alerta e rota da V3.1.

Depois de aplicar, rode git add -A, commit e push.
